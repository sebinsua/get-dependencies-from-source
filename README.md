# `get-dependencies-from-source`

## Usage

```sh
$ get-dependencies-from-source node_modules/precinct/**/*.js
[ 'commander',
  'module-definition',
  'debug',
  'node-source-walk',
  'detective-cjs',
  'detective-amd',
  'detective-es6',
  'detective-less',
  'detective-sass',
  'detective-scss',
  'detective-stylus',
  'detective-typescript' ]
```

## Install

```sh
yarn add get-dependencies-from-source
```
